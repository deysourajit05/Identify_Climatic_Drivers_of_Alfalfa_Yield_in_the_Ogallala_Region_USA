# Identify_Climatic_Drivers_of_Alfalfa_Yield_in_the_Ogallala_Region_USA
This is a GitHub Repo has the data for climatic features collected from PRISM from 1981 to 2018 and county level alfalfa yield data collected from NASS. mRMR technique has been used to make a subset of features, and SHAP values were used to interpret the importance of features in yield estimation.
